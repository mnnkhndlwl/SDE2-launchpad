## CSS selectors

```
#id  .class  tag  tag.class
[name='q']  [href^='https']  [src$='.png']  [class*='btn']
a b (descendant)  a > b (child)  a + b (next sibling)
li:nth-of-type(2)  :first-child  :last-child
```

## XPath

```
//tag[@attr='val']   //*[@id='x']
//button[text()='Login']   //*[contains(.,'Log')]
(//input)[1]
following-sibling::  parent::  ancestor::  child::
contains()  starts-with()  normalize-space()
```

## Git

```
git clone <url>          git checkout -b feat/x
git add .                git commit -m "msg"
git pull                 git push -u origin feat/x
git merge <b>            git rebase main
git restore <file>       git reset --soft HEAD~1
git revert <hash>        git stash / git stash pop
git log --oneline --graph
```

## Test-runner annotations & assertions

```
TestNG (Java):  @BeforeMethod @Test @AfterMethod  Assert.assertEquals  SoftAssert
JUnit5 (Java):  @BeforeEach @Test @AfterEach  @ParameterizedTest  Assertions.assertEquals
pytest (Py):    def test_x():  @pytest.fixture  @pytest.mark.parametrize  assert
Playwright(JS): test('x', async ({page})=>{})  test.beforeEach  expect(loc).toBeVisible()
```

## HTTP status codes

```
200 OK   201 Created   204 No Content
301/302 Redirect   304 Not Modified
400 Bad Request   401 Unauthorized   403 Forbidden
404 Not Found   409 Conflict   422 Unprocessable   429 Too Many Requests
500 Server Error   502 Bad Gateway   503 Unavailable   504 Timeout
```

## Playwright API

```
page.goto(url)   getByRole('button',{name:'Submit'})
locator.click() / .fill('x') / .check()
expect(loc).toBeVisible() / toHaveText('x')
page.route('**/api', handler)        // mock
context.storageState({path:'auth.json'})  // reuse login
```

## Selenium API

```
driver.get(url)   findElement(By.cssSelector("..."))
.click() / .sendKeys("x") / .getText()
new Select(el).selectByVisibleText("x")
new WebDriverWait(driver,Duration.ofSeconds(10))
   .until(ExpectedConditions.elementToBeClickable(by))
driver.switchTo().frame()/alert()/window(handle)
((JavascriptExecutor)driver).executeScript("...")
```
