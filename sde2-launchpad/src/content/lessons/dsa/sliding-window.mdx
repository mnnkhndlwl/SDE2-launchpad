A sliding window maintains a contiguous range `[left, right]` and slides it across the sequence, expanding and contracting so each element is visited at most a constant number of times → O(n).

## Variable window — expand then shrink

The right pointer grows the window; when the window violates the constraint (e.g. a duplicate), the left pointer shrinks it until it's valid again.

<ArrayViz algo="sliding-window" />

```js
// Longest substring without repeating characters
function lengthOfLongest(s) {
  const lastSeen = new Map();
  let left = 0, best = 0;
  for (let right = 0; right < s.length; right++) {
    const c = s[right];
    if (lastSeen.has(c) && lastSeen.get(c) >= left) {
      left = lastSeen.get(c) + 1;       // shrink past the duplicate
    }
    lastSeen.set(c, right);
    best = Math.max(best, right - left + 1);
  }
  return best;
}
```

The key insight: each pointer only moves **forward**, never back. Even though it looks like a nested process, total work is 2n moves → O(n).

## Fixed window — slide a constant size k

When the window size is fixed, add the entering element and remove the leaving one each step:

```js
// Max sum of any window of size k
function maxSum(nums, k) {
  let sum = 0; for (let i = 0; i < k; i++) sum += nums[i];
  let best = sum;
  for (let i = k; i < nums.length; i++) {
    sum += nums[i] - nums[i - k];   // add new, drop old
    best = Math.max(best, sum);
  }
  return best;
}
```
