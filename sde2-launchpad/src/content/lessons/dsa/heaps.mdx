## Backtracking — generate all / search with pruning

The template is **choose → explore → un-choose**. Build a partial solution, recurse, then undo the choice to try the next option. Prune branches that can't lead to a valid answer.

```js
// Subsets: include or exclude each element
function subsets(nums) {
  const res = [], path = [];
  function backtrack(i) {
    if (i === nums.length) { res.push([...path]); return; }
    path.push(nums[i]);        // choose
    backtrack(i + 1);          // explore
    path.pop();                // un-choose
    backtrack(i + 1);          // explore without it
  }
  backtrack(0);
  return res;
}
```

The shape is identical for permutations, combination sum, word search, and N-queens — only the choices and the pruning condition change.

## Intervals — sort then sweep

Almost every interval problem starts by **sorting by start time**, then sweeping left to right and merging/comparing adjacent intervals.

```js
// Merge Intervals
function merge(intervals) {
  intervals.sort((a, b) => a[0] - b[0]);
  const out = [intervals[0]];
  for (const [s, e] of intervals.slice(1)) {
    const last = out[out.length - 1];
    if (s <= last[1]) last[1] = Math.max(last[1], e); // overlap → extend
    else out.push([s, e]);                            // gap → new interval
  }
  return out;
}
```
